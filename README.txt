

ID3 Decision Tree Algorithm

If you are looking to run the program, navigate to main.java, compile, then run its main method.

If you wish to change the Grades.csv file, navigate to the utility package, then to csvReader.java.
Here, you can change the csvFilename variable to the path of the new grades file.

It is assumed that the cells in the final column must either be Yes or No
It is assumed that all grades are capitalized
